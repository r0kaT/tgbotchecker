# Installing
1. Clone repository https://github.com/r0kaT/tgbotchecker.git or Download ZIP
2. Unzip it and edit a "config.py" file
3. Run in a cmd "pip install asyncio logging aiogram requests"
4. Run "main.py"

## Available checkers:
- JUP checker
- KiloEx checker
- Hyperlane checker
